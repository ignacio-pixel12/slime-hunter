# Slime Hunter

## Integrantes

- Ignacio García Belvedere

## Descripción

Slime Hunter es un videojuego de acción y aventura donde el jugador controls a un caballero que debe enfrentar oleadas de slimes en diferentes escenarios. El juego cuenta con múltiples tipos de slimes, zonas con temáticas distintas (bosque, desierto, ruinas, castillo) y un boss secreto.

## Tecnologías

- Java 26
- libGDX (framework de desarrollo de videojuegos)
- Plataforma: Escritorio (LWJGL3)

## Cómo compilar y ejecutar

```bash
# Clonar el repositorio
git clone https://github.com/ignacio-pixel12/slime-hunter.git

# Entrar a la carpeta del proyecto
cd slime-hunter

# Ejecutar (desde la carpeta del proyecto)
./gradlew lwjgl3:run
```
